# Tech Stack
- Websocket
- React.js
- Vite
- Node.js
- Typescript

# How To Start
first, start websocket server at 3000 port
```bash
yarn server
```
second, run your vite project
```bash
yarn vite
```
open http://localhost:5173/ on your browser, enjoy it
